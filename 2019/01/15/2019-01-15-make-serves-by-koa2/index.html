初步计划：

  1. 安装本地版本，边学变做，文章：Koa2进阶学习笔记

  2. 建立本地 git 仓库，建立个人服务器的 远程仓库，每次通过 git 推送完成代码传输(失败，可以推送，但.git 后缀名的文件夹npm无法安装依赖)

  3. 使用 pm2 守护服务进程。
