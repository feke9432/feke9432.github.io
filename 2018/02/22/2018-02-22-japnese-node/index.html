日语学习笔记

发表于 2018-02-22 | 分类于 日记文章

为了秋叶原！

在开这篇笔记之前，其实已经花了很多时间背诵五十音图，至于学习目的，一个是兴趣，另一个就是有打算到日本工作定居的意图，从很小时候的《EVA》开始到现在每季度的番剧，还有任天堂、卡普空的游戏，我是个普通的宅男，所以想宅在日本。

一，资料准备

  • 书籍资料：中日交流标准日本语（新版初级上下册）

  • 视频资料：YOUTUBE的新标日视频
